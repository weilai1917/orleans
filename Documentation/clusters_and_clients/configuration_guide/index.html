    Orleans配置指南

    本配置指南解释了关键配置参数以及它们在大多数典型使用场景中的使用方式。

    orleans可以用于各种适合不同使用场景的配置，例如用于开发和测试的本地单节点部署、服务器群集、多实例azure工作者角色等。

    本指南提供了在目标场景中运行Orleans所需的关键配置参数的说明。还有一些其他的配置参数，主要有助于微调Orleans以获得更好的性能。
